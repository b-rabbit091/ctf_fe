# CTF Frontend Project

This is a **CTF (Capture The Flag)** frontend project built using **React** and **Vite**.

---

##  Installation and Setup

Follow the steps below to install dependencies and run the project in your local environment.

---

### 1. Environment Setup

Please ensure you have the following installed:

- **Node.js:** `v22.21.0`
- **npm:** `v10.9.4`

You can verify your environment versions by running:
```bash
node -v
npm -v
```

### 2. Clone the Repository
```bash
git clone git@github.com:b-rabbit091/ctf_fe.git
cd ctf_fe/
```

### 3. Install Dependencies
```bash 
npm install
```

### 4. Run in Development Environment
```bash 
npm run dev
```

Once the server starts, open your browser and visit:  
[http://localhost:5173](http://localhost:5173)
